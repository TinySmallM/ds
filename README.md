# eRA Project

Интерфейс генерации контента с real-time мок-движком очереди задач на стекe **React + Zustand + TypeScript + Tailwind CSS**.

## 🚀 Быстрый старт
##### Страница с очередью по пути: `/tqueue`

### 1. Клонирование репозитория
```bash
git clone <url-репозитория>
```

### 2. Установка зависимостей
Проект использует пакетный менеджер **Yarn**:
```bash
yarn install
```

### 3. Настройка переменных окружения (Env)
Для запуска проекта **обязательно** требуется валидный секретный ключ авторизации движка. Без него проект заблокирует работу.

1. Создайте файл `.env` в корневом каталоге проекта:
   ```bash
   touch .env
   ```
2. Скопируйте структуру из `.env` и вставьте ваш ключ:
   ```env
   VITE_SECRET_KEY=sidKey
   ```

### 4. Запуск сервера разработки
```bash
yarn dev
```

## 🛠 Скрипты

* `yarn dev` — Запуск локального сервера разработки с Hot Reload.
* `yarn build` — Сборка оптимизированного Production-бандла (упадет, если в окружении нет `VITE_SECRET_KEY`).
* `yarn lint` — Проверка кода линтером (ESLint).
